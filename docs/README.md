# OctoAcme Project Management Docs

Welcome! This README provides an overview of OctoAcme's project management processes and links to detailed process documents.

## Summary of OctoAcme Project Management Processes
OctoAcme delivers projects using a transparent, iterative, and data-informed approach tailored for cross-functional teams. Key practices include:

- **Lifecycle stages:** Initiation → Planning → Execution → Release → Retrospective
- **Core principles:** Customer-first, iterative delivery, clear ownership, and psychological safety
- **Roles:** Project Manager (PM), Product Manager (PdM), Developers, QA/Testing, Stakeholders
- **Artifacts:** Project One-pager, Roadmap, Backlog, Acceptance Criteria, Risk Register, Release Plan
- **Communication:** Regular standups, weekly syncs, milestone reviews, stakeholder updates
- **Quality:** CI automation, code review, risk management, and continuous improvement
